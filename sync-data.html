Sync Portfolio Data

Step 1: Export from Admin

Go to the admin page and click "Export Data" to download your portfolio.json file.

Open Admin Page

Step 2: Replace portfolio.json

Replace the content of data/portfolio.json with your exported data.

File location: /Users/apple/Desktop/Ihjas/data/portfolio.json

⚠️ Important Note

This admin interface saves data to your browser's localStorage. To make changes permanent, you must export the data and manually replace the portfolio.json file. In production, you can set up a backend to handle this automatically.
